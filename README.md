# CRUD Angular + Spring

Projeto de estudo desenvolvido utilizando Angular no frontend e integração preparada para backend Spring Boot.

## 📌 Sobre o projeto

Este projeto consiste em um sistema CRUD (Create, Read, Update e Delete) desenvolvido com Angular. Atualmente, o projeto possui estrutura modular, componentes organizados e um serviço responsável pelo gerenciamento dos dados de cursos.

O objetivo do projeto é praticar:

* Estruturação de aplicações Angular
* Componentização
* Rotas e módulos
* Consumo de dados
* Organização de serviços
* Integração futura com Spring Boot
* Angular Material

## ⚙️ Como executar o projeto

### 1. Clone o repositório

```bash
git clone <URL_DO_REPOSITORIO>
```

### 2. Acesse a pasta do projeto

```bash
cd Crud-Angular-spring
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
ng serve
```

ou

```bash
npm start
```

### 5. Abra no navegador

```bash
http://localhost:4200
```
